crop(component='images', origin=None, shape=None)¶

Crop all images in the batch :param component: string - a component name which data should be cropped :param origin: can be one of:

  • tuple - a starting point in the form of (x, y)
  • CROP_00 - to crop from left top edge (0,0)
  • CROP_CENTER - to crop from center of each image
Parameters:shape – tuple - a crop size in the form of (width, height)

Crop all images to a given shape and a random origin :param component: string - a component name which data should be cropped :param shape: tuple - a crop size in the form of (width, height)

Origin will be chosen at random to fit the required shape

random_rotate(ix, component='images', p=1.0, angle=None, **kwargs)¶

Rotate each image in the batch at a random angle :param component: string - a component name which data should be rotated :param p: float - a probability to apply rotate

(0. - don’t rotate, .5 - rotate half of images, 1 - rotate all images)
Parameters:angle – tuple - an angle range in the form of (min_angle, max_angle), in radians
random_scale(ix, component='images', p=1.0, factor=None, preserve_shape=True, crop=-1)¶

Scale the content of each image in the batch with a random scale factor :param component: string - a component name :param p: float - a probability to apply scale

(0. - don’t scale, .5 - scale half of images, 1 - scale all images)
Parameters:factor – tuple - min and max scale; the scale factor for each image will be sampled from the uniform distribution
